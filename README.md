## 🔐 SecurePass Generator

Generatore di password sicure 100% client-side. Usa `crypto.getRandomValues()` quindi è sicuro.

### Features
- Lunghezza 4-64 caratteri
- Toggle: maiuscole, minuscole, numeri, simboli 
- Escludi caratteri ambigui: I, l, 1, O, 0
- Strength meter basato su entropy in bit
- Copia 1-click
- UI Dark mode + responsive

### Avvio
```bash
npm install
npm run dev
